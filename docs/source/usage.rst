Usage
=====

.. _installation:

Installation
------------

To use ipidea-proxy, first install it using pip:

.. code-block:: console

   (.venv) $ pip install ipidea-proxy

Creating recipes
----------------

For example:

>>> from ipidea_proxy import IpideaProxy

>>> ipp = IpideaProxy()

>>> ipp = IpideaProxy(uid='xxxx', appkey='xxxxxx')
