Welcome to Ipidea-Proxy's documentation!
===================================

**ipidea-proxy** is the Python client library for the IPIdea Proxy Service
that provides IP proxy services around the globe

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
